## NexaLife v0.1.0

首个公开开源版本，提供 NexaLife 的模块化个人管理基础能力。

### ✨ Highlights

- 模块化信息管理：Inbox / Execution / Lifestyle / Vitals / Knowledge / Dashboard
- SwiftUI 原生界面与基础交互流程
- 可选 AI 辅助能力（分类、总结、建议）
- API Token 安全存储（默认 Keychain）
- 应用图标资源更新与发布基础文档完善

### 🔐 Open Source & License

- 仓库已补齐开源基础文档：README / CONTRIBUTING / CODE_OF_CONDUCT / CHANGELOG
- 项目许可证：**GNU GPL v3.0**

### ⚠️ Notes

- 某些第三方能力（如 OAuth 参数）仍为开发占位配置，需在部署前替换为正式配置。
- 请勿提交任何本地密钥、证书或私有配置文件。
