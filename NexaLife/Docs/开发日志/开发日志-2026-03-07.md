# NexaLife 开发日志（2026-03-07）

- 日期：2026-03-07
- 规则：当日仅保留一个开发日志文件；后续修改均追加到本文件。

## 更新记录 1：Execution 中栏交互重排与性能细节优化

### 目标与问题
1. “项目管理”按钮在中栏位置不够直觉，需要贴近短/中/长期项目计数。
2. 搜索输入在数据量增大时需要减少无效重算。

### 关键实现
- `Views/Execution/ExecutionView.swift`
  - 将“项目管理”按钮从独立 action 区移动到 `projectOverviewStrip` 计数行右侧，与短/中/长期统计同一视觉组。
  - 增加搜索防抖状态（`debouncedSearchText` + 可取消 Task），搜索词输入后延迟过滤，降低列表高频重算。

## 更新记录 2：Settings 菜单重复项修复

### 目标与问题
1. App 菜单中出现两个 `Settings...`，影响一致性与系统直觉。

### 关键实现
- `App/NexaLifeApp.swift`
  - 移除手动替换 `.appSettings` 的命令注入，仅保留系统默认 Settings/Preferences 菜单入口。
  - 结果：菜单项恢复单一来源，并随系统语言自动展示。

## 更新记录 3：P0 安全加固（Token 存储与日志）

### 目标与问题
1. API Token 需要默认走 Keychain，并兼容历史本地文件存储迁移。
2. 日志输出需要统一分级并避免敏感信息泄露风险。

### 关键实现
- `Utilities/AppLogger.swift`
  - 新增统一日志入口（error/warning/info/debug）和敏感内容脱敏工具。
- `Services/KeychainHelper.swift`
  - Keychain 存取删补齐状态检查与错误日志，避免静默失败。
- `Services/AICredentialStore.swift`
  - 增加 `bootstrapSecurityDefaults()`：首次启动自动完成“历史本地 token -> Keychain”迁移。
  - 增加 `updateStorageMode(_:)`：切换存储模式时自动迁移已有 token，避免切换后“token 丢失感”。
  - 增加 `clearAPIKey()`：统一清理 Keychain 与本地 token 文件。
- 日志替换覆盖：
  - `Services/AIService.swift`
  - `Services/OAuthService.swift`
  - `Views/Onboarding/LoginView.swift`
  - `Views/Settings/AccountDetailView.swift`
  - `App/AppState.swift`

## 更新记录 4：P0 数据安全（导入快照与失败回滚）+ P1 文案补齐

### 目标与问题
1. 数据导入流程需要可回滚，避免半导入导致状态污染。
2. 新增流程文案需同步进入 `xcstrings`，避免裸 key。

### 关键实现
- `Views/Settings/PreferencesView.swift`
  - 导入前创建快照（`createImportSnapshot`）。
  - 导入异常时自动执行回滚（`restoreImportSnapshot`）。
  - 新增空目录导入校验错误（`DataImportError.emptySource`）。
- `Localizable.xcstrings`
  - 补齐导入回滚相关多语言键（中/英）：
    - `preferences.data.import.error.empty_source`
    - `preferences.data.import.failure.rollback`
    - `preferences.data.import.failure.rolled_back`

## 更新记录 5：Data 页新增“清空过去，重新开始”分区

### 目标与问题
1. 在设置中提供“清空内容 / 删除账号”的重置能力，承载“清空过去，重新开始”的喻义表达。
2. 危险操作需有二次确认与明确反馈。

### 关键实现
- `Views/Settings/PreferencesView.swift`
  - Data 页新增 `preferences.data.section.fresh_start` 分区与说明文案。
  - 新增两个危险操作按钮并接入二次确认：
    - 清空内容（保留账号）
    - 删除账号（清空本地内容并回到 onboarding）
  - 新增本地清理流程：
    - 清空数据目录、备份目录、导入快照目录
    - 清理 `default.store*` 相关本地持久化文件
    - 清理头像路径
  - 新增操作结果状态提示（成功/部分失败）与重启建议。
- `App/AppState.swift`
  - 新增 `resetAccountForFreshStart()`，统一重置账号与偏好为初始状态并回到引导流程。
- `Services/AICredentialStore.swift`
  - 删除账号流程复用 `clearAPIKey()` 清理凭据。
- `Localizable.xcstrings`
  - 补齐 fresh start 分区及确认弹窗文案键。
